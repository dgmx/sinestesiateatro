# Sinestesia Teatro Negro

Static website for a theater company. No build system.

## Structure
- `index.html` — Main page (homepage)
- `last.html` — Previous performances
- `estilo.css` — Styles
- `poster/` — Downloadable posters

## Deployment
Hosted on GitHub Pages. The `CNAME` file maps to `www.sinestesiateatro.es`.

## Notes
- No build tools, tests, or linting
- Just static HTML/CSS/images
- Deploys automatically on push to main
